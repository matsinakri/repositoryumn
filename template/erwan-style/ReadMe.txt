Template Gratis Untuk SLiMS 8 Akasia + Plugin TV Display
Nama template : erwan-style
Modifikator : Erwan Setyo Budi (erwans818@gmail.com)


Fitur Tem:
1. Bukplateu Terbaru
2. Slider berita terbaru
3. Kotak Pencarian statis di atas
4. Menu Drop Down
5. E-Resources6. Gallery
7. Sampel Pilihan warna
8. Display Jumlah tipe koleksi
9. Custom halaman visitor counter terdapat angka jumlah kunjungan
10. TV Display

Fitur display :
running text, tanggal dan waktu real time, buku terbaru, 5 anggota teraktif, statistik, auto refresh

Cara pasang Template:
1. Salin folder "erwan-style" ke folder slims anda/template
2. Salin file "gallery.inc.php" dan tv.inc.php ke folder slims anda/lib/content

Cara akses tv display
1. Sipakan tv dengan akses internet atau Komputer yang layarnya disambungkan ke tv
2. Menujulah ke alamat slims anda, lalu tambhkan pada url index.php?p=tv
contoh : http://localhost/slims/index.php?p=tv

Cara mengaktifkan running text pada tv display
1. Login ke slims, lalu pilih menu system, pilih sub menu content.
2. kemudian buatlah konten baru dengan klik tombol "add new content", lalu lengkapi sebagai berikut:

Content Title*	:	newsticker
This is News	:	YES
Path (Must be unique)*	: newsticker	
Content Description : isi running text yang anda inginkan
3. Simpan
4. Menujulah ke url index.php?p=tv

#Cara isi foto gallery, ganti dengan foto anda pada folder slims anda/template/erwan-style/img/gallery
#Cara ganti warna header : 
1.Buka file index_template.inc.php pada folder slims anda/template/erwan-style
2. Lihat pada baris 52, maka terapat kodeitemscope="itemscope" itemtype="http://schema.org/WebPage" data-ma-theme="blue"

kata "blue" bisa diganti dengan green,blue, red, orange, teal, cyan, blue-grey, purple, indigo, dan brownLalu simpan.
